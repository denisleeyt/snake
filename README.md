## Development Guideline

Install dependencies,

```bash
$ yarn install
```

Start the dev server,

```bash
$ yarn start
```

Build documentation,

```bash
$ yarn docs:build
```

Build documentation and create zip package `snake-docs.zip`,

```bash
$ yarn docs:zip
```

Run test,

```bash
$ yarn test
```

Build library via `father-build`,

```bash
$ yarn build
```

Update CHANGELOG and bump package version manually,

```bash
$ yarn release
```
