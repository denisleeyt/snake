## icon

除了支持在线的图片地址外，也支持所有的 [antd icon](https://ant.design/components/icon-cn/)，命名规则为 Kebab case，单词间以 `-`（连字符）做间隔。

## API

| 参数  | 说明                                                  | 类型   | 默认值 |
